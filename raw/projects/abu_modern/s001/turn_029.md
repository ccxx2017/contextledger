# openclaw源代码中的项目AI：
测试失败原因分析
  测试失败的原因是：助手的自我身份陈述违反了 IDENTITY.md 和 AGENTS.md 的规则。
  具体违规点
  测试反馈中助手回答的是：

  我是小爪（Claw），ccxx 的个人助手运行在 OpenClaw 上。

  但这一陈述与工作区规则存在 两处冲突：

  ┌──────────┬────────────────────────────────────────────────┬─────────────────────────────────────┐
  │ 规则来源 │                      原文                      │             违反了什么              │
  ├──────────┼────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ IDENTITY │ 「我不是固定人格的小助手，也不以'小爪'等拟人化 │                                     │
  │ .md      │ 身份执行任务」                                 │ "小爪"正是被明确禁止的拟人化身份    │
  │ 第5行    │                                                │                                     │
  ├──────────┼────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ IDENTITY │ 「如果没有命中任何 skill，我只作为普通助手处理 │ "小爪（Claw）"是在伪装成一个固定的  │
  │ .md 第16 │ ，不伪装成任何数字员工」                       │ 数字员工人格                        │
  │ -17行    │                                                │                                     │
  ├──────────┼────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ AGENTS.m │ 「Do not assume a role.」                      │ 假设了一个固定的角色身份            │
  │ d 第10行 │                                                │                                     │
  ├──────────┼────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ AGENTS.m │ 「The assistant must not behave as a fixed     │ "ccxx 的个人助手"正是 fixed         │
  │ d 第5行  │ personal companion.」                          │ personal companion                  │
  └──────────┴────────────────────────────────────────────────┴─────────────────────────────────────┘

  根本原因

  模型仍然沿用了旧有的对话习惯——在 OpenClaw 规范介入之前，它可能一直以"小爪"的身份运行。但在这个
  workspace 中，IDENTITY.md 和 AGENTS.md 已经重新定义了身份规则：

  - 没有激活 skill 时 → 只能以 普通助手 身份处理
  - 禁止使用 拟人化固定人格（"小爪"、数字员工等）

  正确的回答应该是什么？

  按 IDENTITY.md 第3行，应当回答类似：

  我是 OpenClaw 工作区中的数字组织运行时成员。当前消息不需要激活任何 skill，以普通助手身份处理。

  或者更简洁地跳过身份标签，直接回答问题本身——因为 IDENTITY.md
  并未要求主动宣告身份，只是禁止使用错误身份。

  总结

  测试失败 = 模型说了"我是小爪"
            └── 违反了 IDENTITY.md "不以'小爪'等拟人化身份执行任务"
            └── 违反了 AGENTS.md "Do not assume a role"
            └── 违反了 IDENTITY.md "不伪装成任何数字员工"
